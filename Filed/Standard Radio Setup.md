There are many correct radio setups you might use.

Jack always recommends this for pre-solo students or pattern work:

| Radio  | Active | Standby |
| ------ | ------ | ------- |
| Comm 1 | TWR    | 121.5   |
| Comm 2 | GND    | ATIS    |

As you start into cross-countries, you may use comm 2 for listening and comm 1 for talking/listening.^[Said differently, using comm 2 for only ATIS or similar]
